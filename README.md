# Digital-Gateway-Key-on-Solana-DGKS

## DGK-S - Digital Gateway Key on Solana

DGK-S is the Solana-native evolution of DGK (Digital Gateway Key).

Total Supply: 25,000,000 DGK-S
Decimals: 9
Network: Solana Mainnet
Standard: SPL Token

This repository contains:
- Whitepaper (EN/ES)
- Token creation scripts (Solana CLI)
- Basic Web interface
- Technical explanation document
